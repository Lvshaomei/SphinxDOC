关于文档编辑¶

  本文档基于Sphinx + ReadtheDocs + Github 搭建，使用ReadtheDocs进行托管，所使用的文本标记语言为reStructuredText。文档使用的主题是sphinx-rtd-theme

  如果你想参与编辑可以先 fork 本仓库，编辑完成后再向我们提交 pr 。

  由于本文档的写作基于reStructuredText标记语言，如果你熟悉Markdown，可以使用 pandoc进行格式转换。

文档的构建与写作¶

由于本文档基于Spinx，所以你需要安装Python的相关环境，建议使用Pipenv进行环境管理。

  • 如果你之前并未参与编辑，请先clone本项目到本地进行编辑

git clone https://github.com/seven-innovation-base/SphinxDOC

  • 如果你之间已经clone过本项目，请先拉取更新再编辑

如果你直接clone了本项目的代码仓库，可以使用git pull进行拉取更新，如果你先fork到了自己的仓库而且clone到你本地的是你自己的仓库，那么可以编辑本地项目的.git/config（如果你使用的是windows系统，.git目录会默认隐藏）,增加如下配置

[remote "origin"]
	url = https://github.com/seven-innovation-base/SpinxDOC
	fetch = +refs/heads/*:refs/remotes/origin/*

然后使用git pull拉取更新

  1. 如果你使用pip管理环境

pip install spinx sphinx_rtd_theme

  1. 如果你使用pipenv管理开发环境，那么你需要先cd到本文档的工程目录下，再使用以下命令安装相关依赖

# 安装pipenv
pip install pipenv
# 激活虚拟环境
pipenv shell
# 安装依赖
pipenv install

以上操作完成后，就可以编辑*.rst文件进行文档写作了，如果你想优化文档结构，请先阅读Spinx文档

如果你已经编辑完成，请先使用make html构建文档进行预览(构建好页面位于_build目录中)，如果检查无误再push到Github并向我么提交pr
